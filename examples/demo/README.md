## Kamera demo

```console
go run github.com/setanarut/kamera/examples/demo@latest
```

Run example on your local machine

| Key       | Action                     |
| -----     | -------------------------- |
| WASD      | Move camera                |
| T         | Add 0.6 Trauma             |
| E         | Zoom in                    |
| Q         | Zoom out                   |
| Backspace | Reset Rotation/Zoom        |
| R         | Rotate                     |
| X         | Look at the random object. |
| L         | Toggle Lerp                |
