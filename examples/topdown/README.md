## `kamera.Camera` top down example

### Keyboard Controls

| Key   | Action                     |
| ----- | -------------------------- |
| WASD  | Move camera                |
| E     | Zoom in                    |
| Q     | Zoom out                   |
| Space | Reset camera               |
| Key 1 | look at the random object. |
| Key 2 | Reset Zoom                 |
